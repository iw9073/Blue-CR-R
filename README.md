# Blue-CR-R
Features:
.Pairing and unpairing Bluetooth devices.
.Connection and Disconnection of devices.
.Power Control over connected devices.
Ie: Power off.

.I also “Want” to the app to be able to display battery percent for certain devices like
wireless headphones.
In function I want a user to be able to in a simple manner of three clicks or less connect
or disconnect any paired Bluetooth device, pair, or unpair a device or power off a
connected device.
